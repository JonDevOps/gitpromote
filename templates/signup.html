Gitpromote

Promote your open source projects to many other people like you!

Note: This app is in the initial stage of development. It may not run successfully in other browsers except Google Chrome

Join the growing community of github users!

How gitpromote works:

1. gitpromote takes information about your public repositories from github and creates your profile.

2. Every gitpromote user initially gets 200 points as a token of appreciation.

3. A user can promote his repository to any number of people he/she wants with the cost of points which makes your repository reach more people.

4. gitpromote algorithm works in such a way that it promotes the repository to the people who have got some good amount of contributions on github in the language of the repository you are promoting.

5. By promoting your repository on gipromote, users like you gain the sight of your repository so that you'll have the right people to add new features and to improve the code.

So join the growing community of open source users to help others thereby helping yourself ;)
